# TransitionIQ - Enterprise Hub

TransitionIQ is an intelligent dashboard built to streamline employee transitions (onboarding and offboarding), manage hardware logistics, capture institutional knowledge, and audit software licenses. 

This project is built using a lightweight **Flask (Python)** backend and a vanilla JavaScript frontend utilizing **Tailwind CSS** and **Lucide Icons**.

## 📁 Project Structure

Ensure your project files are organized exactly like this before running the application:

```text
